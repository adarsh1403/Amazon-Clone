# Amazon Clone

## Overview
A responsive, static front-end clone of the Amazon e-commerce website interface built purely with HTML5 and CSS3. This project aims to replicate the core visual structure and layout of Amazon's homepage, demonstrating proficiency in web design, CSS layout techniques (like Flexbox), and UI recreation.

## Features
- **Header & Navigation:** Includes the Amazon logo, location selector, a flexible search bar with a dropdown menu, sign-in section, and shopping cart indicator.
- **Secondary Navigation Panel:** A sub-navigation bar with links to categories like "Today's Deals", "Customer Service", and more.
- **Hero Section:** A prominent banner mimicking Amazon's homepage promotions with an embedded link.
- **Product Grid:** A responsive CSS flexbox layout displaying various product categories (e.g., Clothes, Electronics, Furniture, Pet Care) with images.
- **Detailed Footer:** A comprehensive replica of Amazon's multi-column footer containing links to services, business info, payment products, and a bottom panel with copyright details.
- **Iconography:** Integrated with Font Awesome for authentic-looking UI icons (magnifying glass, cart, location dot, etc).

## Technologies Used
- **HTML5:** Semantic structuring of the webpage.
- **CSS3:** Styling, custom variables, and Flexbox for responsive component layout.
- **Font Awesome 7.0.1:** External CDN for scalable vector icons.

## Project Structure
```text
Amazon-Clone/
├── assets/          # Directory containing banner and product preview images
├── index.html       # The main HTML structure and layout
├── style.css        # The CSS styling and layout rules
└── README.md        # Project documentation
```

## How to Run
1. Clone the repository or extract the project folder.
2. Navigate to the project directory.
3. Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, Safari) to view the project. 
*Note: No server setup, Node.js, or additional dependencies are required since it is a static site.*
